Documenting my journey of setting up Minecraft on Ubuntu (Linux), with (hopefully) useful mods.

Writing this for anyone who wants to copy my methods -- which is most likely my future-self, more than anyone else ;)

Get a PC, install Ubuntu

First, get a new PC, that is so cheap it will make Windows painfully slow -- but linux, being actually good, works great on it!

Grab a blank 16 GB USB flash drive (only needs 6 GB allegedly), and follow this process to make a bootable USB:
https://ubuntu.com/tutorials/create-a-usb-stick-on-ubuntu#1-overview

Edit your BIOS settings to allow booting from USB. (the above does not require disabling UEFI)

Install Ubuntu from the disk. Boot into your shiny new linux; mess around a bit.

Set up your browser, install extensions as needed.

Install apps

IntelliJ

Open the App Center and install: "IntelliJ IDEA Community Edition" by jetbrains.
Later on, wonder why you used the App Center instead of their own installation guide:
https://www.jetbrains.com/help/idea/installation-guide.html

git, gimp, auotkey

Open the terminal (CTRL+ALT+T) and install a few apps:

  ~$ sudo apt install git gimp autokey-gtk

(later learn that AutoKey doesn't work at the moment; be sad and hope for a better solution later)

Minecraft

For Minecraft itself, use the official website.

  1. Go to: https://www.minecraft.net/en-us/download
  2. Download the "DEBIAN + DEBIAN BASED" option.

Open the terminal again. 

~$ cd ~/Downloads
~/Downloads$ sudo dpkg -i Minecraft.deb

Install Mod Loaders

First download Forge and LiteLoader:

  1. Open https://files.minecraftforge.net/net/minecraftforge/forge/index_1.12.2.html
  2. click "Installer" to download the jar file; wait for ad then click SKIP
  3. Open https://www.liteloader.com/download
  4. click 1.12.2 SNAPSHOT
  5. click Download JAR
  6. You should now have two files in your ~/Download folder, similar to:
    • forge-1.12.2-14.23.5.2860-installer.jar
    • liteloader-installer-1.12.2-00-SNAPSHOT.jar
    You will need the exact names next.
  7. Open the terminal again, and run the following commands. Ensure you have the exact jar file names from the downloads. 
    sudo apt-get update
sudo apt --fix-broken install
sudo apt install openjdk-8-jdk openjdk-8-jre -y
cd ~/Downloads
java -jar forge-1.12.2-14.23.5.2860-installer.jar nogui
  8. click "Install Client"; when get Complete message click OK
  9. Back in the terminal 
    java -jar liteloader-installer-1.12.2-00-SNAPSHOT.jar nogui
  10. in "Extend from" select the new forge installation, click OK.

Download / Install Mods

Mod Source File Name Description
AntiGhost CurseForge antighost-1.12-forge14.21.1.2387-1.1.1.jar Fix ghost blocks with a key-press, so you can spend less time re-logging.
Xaeros World Map CurseForge XaerosWorldMap_1.39.12_Forge_1.12.jar Good world-map mod.
Xaeros Minimap CurseForge Xaeros_Minimap_FP22.2.0.1_Forge_1.12.jar Good mini-map mod. This version is MCO-safe by default, by not showing any entities or caves.
OptiFine optifine.net OptiFine_1.12.2_HD_U_G5.jar Improves FPS (render time), gives good options; this is also required to see custom MCO textures.
ColorUtility CurseForge ColorUtility-universal-1.0.4.jar Write formatted text on signs and books.
MaLiLib modrinth.com malilib-forge-1.12.2-0.10.0-dev.23.jar Required by Litematica.
Litematica modrinth.com litematica-forge-1.12.2-0.0.0-dev.20191217.195055.jar For saving builds made in Creative, and helping build them in MCO. Saves *lots* of time on large projects.
WorldEdit CurseForge worldedit-forge-mc1.12.2-6.1.10-dist.jar In-game Minecraft map editor. For designing builds in Creative, to then use with Litematica.
Ghostwriter github 1.12.2-Ghostwriter-1.9.8.jar Utility for dealing with books (backup, restore, copy, paste, etc.)
Named Heads CurseForge namedheads-0.2.0+1.12.2.jar Shows the player-name of a placed player-head block, similar to renamed item in an item frame.

After downloading all mods, move all those jar files from your ~/Downloads folder to:

  • ~/.minecraft/mods/1.12.2

Configure Minecraft & Mods

First correct the many bad defaults in vanilla Minecraft 1.12.2.

  • FOV: 90
  • Realms Notifications: OFF
  • Music & Sounds...
    • Show Subtitles: ON
    • (leave sound levels; mute in the operating system not in settings)
  • Video Settings...
    • Graphics: Fast
    • Render Distance: 12
    • Smooth Lighting: Minimum
    • Smooth Lighting Level: OFF
    • View Bobbing: OFF
    • GUI Scale: Normal
    • Brightness: either +50% or Bright
    • Dynamic Lights: Fast
    • Animations...
      • Particles: Decreased
    • Other...
      • Advanced Tooltips: ON
  • Controls...
    • Auto-Jump: OFF
    • Pick Block: V
    • Xaero's World Map > Open World Map: SHIFT + M (to avoid conflict with Litematica's "M")

  • Xaero's World Map
    • Block Colours: Vanilla

    • Biomes In Vanilla Mode: ON
