Documenting my journey of setting up Minecraft on Ubuntu (Linux), with (hopefully) useful mods.

Writing this for anyone who wants to copy my methods -- which is most likely my future-self, more than anyone else ;)

Get a PC, install Ubuntu

First, get a new PC, that is so cheap it will make Windows painfully slow -- but linux, being actually good, works great on it!

Grab a blank 16 GB USB flash drive (only needs 6 GB allegedly), and follow this process to make a bootable USB:
https://ubuntu.com/tutorials/create-a-usb-stick-on-ubuntu#1-overview

Edit your BIOS settings to allow booting from USB. (the above does not require disabling UEFI)

Install Ubuntu from the disk. Boot into your shiny new linux; mess around a bit.

Set up your browser, install extensions as needed.

Install apps

Open the App Center and install: "IntelliJ IDEA Community Edition" by jetbrains.
Later on, wonder why you used the App Center instead of their own installation guide:
https://www.jetbrains.com/help/idea/installation-guide.html

Open the terminal (CTRL+ALT+T) and install a few apps:

sudo apt install git gimp autokey-gtk

Later learn that AutoKey doesn't work at the moment; be sad and hope for a better solution later

For Minecraft itself, use the official website.

  1. Go to: https://www.minecraft.net/en-us/download
  2. Download the "DEBIAN + DEBIAN BASED" option.

Open the terminal again. 

cd ~/Downloads
sudo dpkg -i Minecraft.deb

Download / Install Mods 

AntiGhost (antighost-1.12-forge14.21.1.2387-1.1.1.jar) - less re-logging for those who like to instant-mine.
https://www.curseforge.com/minecraft/mc-mods/antighost/files/2856128

Xaeros Map (XaerosWorldMap_1.39.12_Forge_1.12.jar, Xaeros_Minimap_FP22.2.0.1_Forge_1.12.jar) - both world-map and mini-map ("fair play") mods.
https://www.curseforge.com/minecraft/mc-mods/xaeros-world-map
https://www.curseforge.com/minecraft/mc-mods/xaeros-minimap-fair-play-edition

OptiFine (OptiFine_1.12.2_HD_U_G5.jar) - required to see custom MCO textures.
https://optifine.net/downloads

ColorUtility (ColorUtility-universal-1.0.4.jar) - write formatted text on signs and books.
https://www.curseforge.com/minecraft/mc-mods/colorutility/files/2440182

Litematica (litematica-forge-1.12.2-0.0.0-dev.20191217.195055.jar, malilib-forge-1.12.2-0.10.0-dev.23.jar) - for saving builds made in Creative, and helping build them in MCO. Saves *lots* of time on large projects.
https://modrinth.com/mod/malilib
https://modrinth.com/mod/litematica/version/0.0.0-dev.20191217.195055

WorldEdit (worldedit-forge-mc1.12.2-6.1.10-dist.jar, WorldEdit-CUI-FE3-1.12.2-3.0.9.jar) - "In-game Minecraft map editor"
https://www.curseforge.com/minecraft/mc-mods/worldedit/files/6693952

move all jar files to:
~/.minecraft/mods/1.12.2
